Задача и данные

Исходные данные

  • 543 МБ JSON — вакансии HeadHunter
  • SQLite база: 2096 вакансий
  • 5 профессий (Категория 4)

Профессии

DevOps SysAdmin Cloud Eng. SRE Network Eng.

Этапы обработки

  • Парсинг JSONL → фильтрация по категориям → загрузка в БД
  • Классификация по названию → извлечение навыков → определение грейда
  • TF-IDF + cosine similarity для поиска релевантных вакансий

Архитектура приложения

Стек технологий

  • Frontend: Streamlit + Plotly
  • Backend: Python 3.11
  • БД: SQLite
  • ML: scikit-learn (TF-IDF)
  • Деплой: Docker

4 блока дашборда

  • Общие сведения о выборке
  • Типовая вакансия по профессии
  • Классы вакансий (обобщение)
  • Поиск по навыкам работника

Ключевые возможности

  • Автоматическая классификация вакансий по 5 профессиям
  • Извлечение зарплаты с фильтрацией ложных срабатываний
  • Разделение описания на Требования / Обязанности / Условия
  • Поиск вакансий по навыкам с расчётом процента совпадения

Запуск приложения

# Клонировать репозиторий
git clone https://github.com/rendonett/tssa_project.git
cd tssa_project

# Собрать Docker-образ
docker build -t vacancy-dashboard .

# Запустить
docker run -p 8501:8501 vacancy-dashboard

# Открыть в браузере
http://localhost:8501

Работает на Windows, Linux, macOS — достаточно установленного Docker
